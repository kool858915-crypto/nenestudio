# NENE Studio UI Wireframe

NENE Studio MVPの静的ワイヤーフレームです。

## 開き方

```bash
npm install
cp .env.example .env
npm run dev
```

ブラウザで `http://localhost:8787` を開くと確認できます。PWAとして使う場合は、ブラウザの「アプリをインストール」または「ホーム画面に追加」から追加します。

独自ドメイン **`nenestudio.net`** の公開手順は `DEPLOYMENT.md` を参照してください（Cloudflare Pages + Render）。

## 出力したコードをツールとして使う手順

1. `ZIP形式` を選んで `ツールを作成する` を押します。
2. 無料プランでは出力前に広告が表示されます。月額780円プランはStripe Webhookで決済完了確認後に広告なしで出力できます。
3. ダウンロードされたZIPを解凍します。
4. 解凍したフォルダの `index.html` をブラウザで開きます。
5. OpenAI APIキーと本文・元情報を入力して使います。

フォルダ形式をコピーして使う場合だけ、`--- index.html ---`、`--- style.css ---`、`--- script.js ---` の区切りごとにファイルへ分けて保存します。

## 入力ルール

- URLを読ませる仕様は禁止です。
- URL欄はメモ用です。本文は自動取得しません。
- 実際に処理する内容は、本文貼り付け欄に入れた文章・元情報です。
- 自動取得が必要な場合は、正式な外部APIキーを入力して使う仕様に限定します。

## 支払い設定

- 支払いはStripeのみです。
- Stripeで月額780円のPriceを作成し、サーバーの `.env` に `STRIPE_PRICE_ID` を設定します。
- 決済方法はStripe側でカード決済と携帯払い（Apple Pay / Google Pay）を有効化します。
- Stripe Webhookの送信先は `/api/stripe/webhook` です。
- 決済完了後はWebhookでDBの課金状態を更新します。URLだけでは広告なしになりません。

## サーバー側で保護するもの

- `OPENAI_API_KEY` は `.env` に保存し、ブラウザには出しません。
- Stripe秘密キー、Webhook署名キー、Price IDも `.env` に保存します。
- ログインユーザー、保存済みツール、課金状態はJSONファイルDBに保存します。
- ブラウザからのAI生成は `/api/ai/generate` を呼び、サーバーがOpenAI APIを実行します。

## 使用例

出力内容に `--- index.html ---`、`--- style.css ---`、`--- script.js ---` が含まれている場合、それぞれを同じ名前のファイルに分けて保存します。同じフォルダに置いたあと `index.html` を開くと、ブラウザでツールとして使えます。
