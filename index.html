このツールは何をする？

ニュースから注目テーマと関連銘柄を整理します。

何を入れればいい？

ニュース本文、見たい市場、出力形式、URLメモを入れます。

何が出てくる？

注目テーマ、関連銘柄、理由、投稿用メモが出ます。

どう使う？

APIキーを設定し、ニュースを指定して実行します。

tool_name: ニュース連動型注目株発見ツール
purpose: ニュースから関連テーマと銘柄を整理する
target_user: 投資メモや投稿案を作りたい初心者
inputs: ニュース本文, URLメモ, 市場, 出力形式
workflow: 取得 -> 抽出 -> 分析 -> 点数化 -> レポート
required_api_keys: OpenAI API key, News API key
folders: config, prompts, workflow, output
error_handling: 本文未入力、APIキー未設定、取得失敗を表示
prompt: ニュースを読み、投資判断ではなく整理情報としてまとめる

作成したツールの使い方

ZIP形式なら、解凍して `index.html` を開くだけで使えます。

フォルダ形式でコピーした場合だけ、区切りごとにファイルへ分けてください。

README index.html script.js

1. ZIP形式で作成する

`ZIP形式` を選んで `ツールを作成する` を押すと、実行用ファイル一式が入ったZIPが出ます。

コピー内容 -> 各ファイル

2. ZIPを解凍する

解凍すると `index.html`、`style.css`、`script.js` などが最初から分かれた状態で入っています。

index.html

3. 起動する

解凍したフォルダの `index.html` をブラウザで開きます。この画面の `作成したツールを起動する` でも確認できます。

APIキー 本文貼り付け 生成する

4. APIキーと本文を入れて使う

URLはメモ用です。実際に処理したい本文や元情報を貼り付け、OpenAI APIキーを入れて生成します。

フォルダ形式をコピーした場合だけ作るファイル

`index.html`

画面本体です。出力内の `--- index.html ---` から次の区切りまでを入れます。

`style.css`

見た目のファイルです。`index.html` と同じフォルダに置きます。

`script.js`

生成ボタンやAPI通信の動きです。`index.html` と同じフォルダに置きます。

補助ファイル

`README.md`、`setup.md`、`prompts/main_prompt.md` などは説明・保管用です。実行に最低限必要なのは上の3つです。
