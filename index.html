このツールは何をする？

ニュースから注目テーマと関連銘柄を整理します。

何を入れればいい？

何が出てくる？

どう使う？

APIキーを設定し、ニュースを指定して実行します。

tool_name: ニュース連動型注目株発見ツール
purpose: ニュースから関連テーマと銘柄を整理する
target_user: 投資メモや投稿案を作りたい初心者
inputs: ニュース本文, URLメモ, 市場, 出力形式
workflow: 取得 -> 抽出 -> 分析 -> 点数化 -> レポート
required_api_keys: OpenAI API key, News API key
folders: config, prompts, workflow, output
error_handling: 本文未入力、APIキー未設定、取得失敗を表示
prompt: ニュースを読み、投資判断ではなく整理情報としてまとめる

Free Plan

無料プラン：設計図を作り、Codex 等で実装する

APIキーは不要です。NENE Studio で設計図を出力し、Cursor（Codex）・Claude Code・ChatGPT などの外部AIに渡して実装します。

無料プランでできること
  • カテゴリ選択 → 提案 → 部品 → 設計書の作成（API不要）
  • Codex用 / Claude Code用 の文案をダウンロード
  • 実装作業は あなたが使う外部AI で行う（NENE Studio の API 代はかかりません）
  • 出力前に広告が表示されます（480円以上の有料プランで広告なし）

手順A：NENE Studio で設計図を作る（約5〜10分）

  1. 左メニューの 「作る」 を押します。
  2. 作りたい カテゴリを1つ選び、右上 「次へ進む」 を押します。
  3. 質問1〜3 はそのままでも、ボタンを変えてもOKです。各画面で 「次へ進む」 を押します。
  4. 「提案を選びます」 で使いたい案を1つ選び、「次へ進む」 を押します。
  5. 「作業の部品を組みます」 で流れを確認し、「次へ進む」 を押します。
  6. 「作成内容を確認します」 → 「詳細表示」 で全文を確認 → 「次へ進む」 を押します。
  7. 「ツールを作成します」 画面で、次のどちらかを選びます。
    • Codex用ツール作成内容 … Cursor（Codex）向け
    • Claude Code用ツール作成内容 … Claude Code 向け
  8. 「ツールを作成する」 を押します（無料プランは広告のあと出力）。
  9. ダウンロードされたファイル、またはコピーした設計書を メモ帳などに保存 しておきます。

手順B：Cursor（Codex）で実装する

Cursor は AI 付きエディタです。設計書を貼り付けるだけで、ファイル作成を依頼できます。

  1. https://cursor.com/ を開き、Cursor をインストールします。
  2. PC上に 新しいフォルダ を作ります（例：my-tool）。
  3. Cursor でそのフォルダを開きます。
  4. チャット（Codex / Agent）を開き、NENE Studio の 設計書全文 を貼り付けます。
  5. 下の 「依頼文例」 をそのまま送ります。
  6. 生成された index.html などをフォルダに保存します。
  7. フォルダ内の index.html をブラウザで開き、動作を確認します。

手順C：Claude Code で実装する

  1. https://claude.ai/ にログインします（Claude Code / ターミナル連携は Anthropic の案内に従います）。
  2. 作業用フォルダを用意します。
  3. NENE Studio の Claude Code用ツール作成内容 を貼り付けます。
  4. 下の 「依頼文例」 を送り、ファイルを作成してもらいます。
  5. できあがったファイルをフォルダに置き、index.html をブラウザで開きます。

手順D：ChatGPT で実装する

  1. https://chatgpt.com/ を開き、ログインします。
  2. 新しいチャットを開始し、NENE Studio の設計書（Codex用または詳細表示の内容）を貼り付けます。
  3. 依頼文例を送り、返ってきた HTML / CSS / JS を ファイルに分けて保存 します。
  4. 同じフォルダに index.html・style.css・script.js を置き、ブラウザで開きます。

以下は NENE Studio で作ったツール設計書です。

設計書：

うまくいかないとき

  • 設計書が短い … 「詳細表示」の内容まで含めて貼り付けてください。
